# trigger-dynamo

Lambda for insert a "Viking" and log information.

## Technologies

- [DynamoDB](https://aws.amazon.com/dynamodb/?nc1=h_ls)

## Installation

**Install Serverless framework**

``` 
    npm i -g serverless
```

**Install dependencies**

```
    npm install
```

## How to run ?

**Deploy**

```
    sls deploy
```

**Invoke function - Local**

``` 
    npm run invoke-local
```

**Invoke function - AWS**

``` 
    npm run invoke
```

**Remove from AWS**

```
    sls remove
```
